## Activitee

Sort your entries by most recent activity (new entry, comments)

#### Tags
##### {exp:activitee:entries}
Works exactly like the {exp:channel:entries} tag.

      <ul>
      {exp:activitee:entries channel='blog'}
        <li>{title}</li>
      {/exp:activitee:entries}
      </ul>
